## Overview

This repository documents projects and learning focused on **business forecasting and predictive analysis**. The goal is to build practical forecasting skills used in business analytics and decision support.

The repository will gradually include multiple forecasting approaches such as scenario analysis, regression models, and time series forecasting.

## Forecasting Techniques

- Revenue forecasting  
- Scenario analysis  
- Regression based forecasting  
- Time series forecasting  
- Forecast evaluation  

## Current Project

### Revenue Forecast Scenario Analysis

This project explores revenue forecasting using structured business assumptions and scenario modeling.

Techniques applied:

- Top down forecasting  
- Bottom up forecasting  
- Optimistic scenario  
- Base scenario  
- Pessimistic scenario
